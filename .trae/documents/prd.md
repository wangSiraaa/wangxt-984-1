## 1. 产品概述

校园校车站点牌、线路调度与学生可乘规则系统——面向调度员、学生、老师、校车管理员的校车智能调度与乘车规则实时推导平台。系统通过本地数据驱动，实现线路/站点/车辆/班次/学生年级/老师规则的多维度联动，支持车辆停运、站点封闭、满员隐藏、故障换车、低年级限制、天气延误、紧急绕行等复杂场景的实时展示与规则推导。

- 目标用户：校园校车调度员、在校学生（按年级区分）、教师、校车管理员
- 核心价值：确保学生乘车安全、校车调度高效、规则透明可解释

## 2. 核心功能

### 2.1 用户角色

| 角色 | 身份说明 | 核心权限 |
|------|----------|----------|
| 调度员 | 校车调度中心工作人员 | 维护线路、站点、车辆、司机、班次、停运、临时绕行 |
| 学生 | 在校学生（按年级1-6区分） | 按年级/站点/身份查看可乘线路、到站预估、换乘提醒 |
| 老师 | 班主任/年级主任 | 维护低年级指定线路、临时停靠、特殊放学安排 |
| 校车管理员 | 车队管理负责人 | 维护车辆满员、故障、替换车辆、年检、驾驶证 |
| 家长 | 学生监护人 | 授权乘车、查看学生乘车状态 |

### 2.2 功能模块

1. **调度端**：线路管理、站点管理、班次管理、司机排班、停运/绕行管理
2. **站点牌大屏**：实时到站预测、线路展示、停运/封闭提示、换乘提醒
3. **学生视图**：可乘线路推导、到站预估、乘车卡、请假停乘、家长授权状态
4. **老师配置**：低年级指定线路、临时停靠规则、特殊放学安排、护送规则
5. **车辆状态**：车辆满员/故障/年检/替换车、司机排班、驾驶证有效期
6. **规则解释面板**：实时推导过程展示、规则命中说明、历史版本
7. **线路模拟器**：模拟停运/绕行/满员/故障场景对站点的影响
8. **应急调度**：天气延误、紧急绕行、车辆故障快速替换
9. **历史回放**：历史调度记录、规则变更历史、乘车记录回溯

## 3. 核心流程

### 3.1 学生可乘线路推导流程

学生查看可乘线路时，系统按以下顺序实时推导：
1. 验证学生年级和身份
2. 检查家长是否已授权
3. 检查学生是否请假停乘
4. 筛选经停学生站点的线路
5. 应用老师指定的低年级线路限制
6. 排除停运线路和封闭站点
7. 排除满员车辆
8. 应用绕行规则更新到站时间
9. 应用临时停靠影响
10. 应用天气延误
11. 输出最终可乘线路列表及到站预估

### 3.2 车辆故障替换流程

1. 管理员标记车辆故障
2. 系统自动查找同线路可用替换车辆
3. 替换车辆接管班次
4. 站点牌和学生视图自动更新
5. 如果无替换车辆，该班次标记为停运

### 3.3 站点封闭换乘流程

1. 调度员标记站点封闭
2. 经停该站点的线路不显示该站到站时间
3. 系统查找最近可用替代站点
4. 站点牌显示换乘提示

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝 (#1E3A5F) + 活力橙 (#FF6B35) + 安全绿 (#2ECC71) + 警告红 (#E74C3C)
- **辅助色**：浅灰背景 (#F5F7FA)、白色卡片、深色文字
- **按钮风格**：圆角 (8px)、扁平化设计、hover时轻微上浮阴影
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式布局
- **图标风格**：Lucide React 线性图标，2px描边
- **动效**：页面切换淡入淡出、列表项交错动画、状态变更脉冲动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 调度端 | 线路管理 | 表格+表单弹窗、站点拖拽排序、线路颜色标识 |
| 调度端 | 班次管理 | 日历视图+时间线、班次状态色块 |
| 站点牌大屏 | 到站预测 | 大字体倒计时、线路颜色条、动画进度条 |
| 站点牌大屏 | 封闭/满员提示 | 红色警告横幅、换乘引导箭头 |
| 学生视图 | 可乘线路 | 卡片式线路列表、绿色可乘/灰色不可乘、到站时间高亮 |
| 学生视图 | 乘车卡 | 虚拟卡片设计、二维码占位、状态标签 |
| 老师配置 | 线路指定 | 左侧年级树+右侧线路勾选、拖拽排序 |
| 车辆状态 | 车辆管理 | 状态色块（绿/黄/红）、替换车连线动画 |
| 线路模拟器 | 场景模拟 | 线路地图可视化、站点节点高亮、时间线推演 |
| 应急调度 | 天气/绕行 | 全屏预警横幅、一键操作按钮、影响范围高亮 |
| 规则解释 | 推导过程 | 步骤流程图、规则命中高亮、绿色通过/红色阻断 |
| 历史回放 | 时间轴 | 底部时间轴滑块、事件气泡、播放/暂停控制 |

### 4.3 响应式设计

- 桌面端优先设计（1920x1080为基准）
- 站点牌大屏适配横屏电视（16:9）
- 学生视图适配移动端（375px起）
- 平板端适配老师配置界面（768px起）

### 4.4 视觉特色

- 线路以不同颜色区分，贯穿所有页面保持一致
- 状态变更使用脉冲动画提示
- 站点牌大屏使用深色背景+高亮文字，确保远距离可读
- 规则解释使用流程图可视化推导链路
- 应急调度使用红色主题+闪烁动画
